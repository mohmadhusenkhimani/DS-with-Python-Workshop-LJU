# 📖 Classwork Programs

## C1 - Attendance Student (Array)

**Topic:** Array (Direct Access)

**Description:**

A class consists of 30 students, where each student's attendance is stored at a fixed index based on their roll number. The program allows the teacher to:

- Mark Attendance (Present/Absent)
- View Attendance Register
- Exit

**Data Structure Used:** Array

---

## C2 - Undo Machine (Stack)

**Topic:** Stack (LIFO)

**Description:**

A simple text editor simulation where every typed character is pushed onto a stack. When the user performs an Undo operation, the last entered character is removed.

Features:

- Type Character
- Undo Last Character
- Display Current Text
- Exit

**Data Structure Used:** Stack

---

## C3 - Smart Printer Queue

**Topic:** Priority Queue

**Description:**

An office printer receives both urgent and normal print jobs. Urgent jobs are always printed before normal jobs.

Features:

- Add Print Job
- Print Next Job
- Display Queue
- Exit

**Data Structure Used:** Two Queues (Urgent & Normal)

# 📝 Homework Programs

## H1 - Amazon Fulfillment Centre

**Topic:** Fixed Array

**Description:**

Simulates a conveyor belt with 8 fixed slots used to store products.

Features:

- Add Product
- Find Product
- Update Product
- Check Belt Status
- Display Products
- Exit

**Data Structure Used:** Array

---

## H2 - Toll Plaza Simulation

**Topic:** Circular Queue

**Description:**

Simulates a toll plaza where vehicles are managed using a Circular Queue to efficiently reuse empty slots.

Features:

- Enqueue Vehicle
- Dequeue Vehicle
- Display Queue
- Exit

**Data Structure Used:** Circular Queue

---

## H3 - GPS Navigation System

**Topic:** Backtracking

**Description:**

Simulates a GPS navigation system where users can visit new locations, move backward to previous locations, and move forward again.

Features:

- Visit New Place
- Go Back
- Go Forward
- Display Current Place
- Exit

**Data Structure Used:** Two Stacks
