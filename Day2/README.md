## 📖 Classwork Programs

### C1 - Bubble Sort
Sort a list of book weights in ascending order using the Bubble Sort algorithm.

### C2 - Selection Sort
Arrange student marks in descending order using the Selection Sort algorithm.

### C3 - Linear Search
Search for a student's name in an unsorted attendance register using Linear Search.

### C4 - Binary Search
Search for a word in a sorted dictionary using the Binary Search algorithm.

---

## 📝 Homework Programs

### H1 - Spam Detector
Detect whether an incoming sender ID exists in a blacklist using Linear Search.

### H2 - E-Commerce Price Filter
Find the first product whose price is greater than or equal to a target price using Binary Search (Lower Bound).

### H3 - IRCTC Waitlist Merger
Sort waitlist numbers using the Merge Sort algorithm.
