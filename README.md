# 🚀 How to Use

1. Clone this repository.

```bash
git clone https://github.com/<your-username>/DS-with-Python-LJU.git
```

2. Open the project in Visual Studio Code.

3. Navigate to the required workshop day.

Example:

```text
Day-01/Classwork/
```

or

```text
Day-02/Homework/
```

4. Run the desired Python program.

```bash
python filename.py
```
